Vetores e Espaços Vetoriais

Um espaço vetorial sobre um corpo 𝔽 é um conjunto V equipado com duas operações — adição de vetores e multiplicação por escalar — que satisfazem os axiomas de fechamento, associatividade, comutatividade e distributividade.

"A geometria de espaços de alta dimensão é contra-intuitiva: quase todo volume de uma hiperesfera concentra-se na superfície." — Blum, Hopcroft & Kannan

Definição formal

Seja V um espaço vetorial de dimensão n. Uma base de V é um conjunto linearmente independente que [[gera V]].

python import numpy as np A = np.array([[1, 2], [3, 4]]) eigenvalues, eigenvectors = np.linalg.eig(A) print(f"Autovalores: {eigenvalues}")

Transformações Lineares

Uma função T: V → W é uma transformação linear se preserva adição vetorial e multiplicação por escalar. Toda transformação linear entre espaços de dimensão finita pode ser representada por uma matriz.

  • O núcleo (kernel) de T é o conjunto de vetores mapeados para zero
  • A imagem é o conjunto de todos os valores possíveis de T(v)
  • Pelo Teorema do Núcleo-Imagem: dim(ker T) + dim(im T) = dim(V)

Ver também: [[Decomposição SVD]] e [[Autovetores e Autovalores]] para aplicações em machine learning.
